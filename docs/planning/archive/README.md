# Historical Planning Archive

This directory preserves completed planning documents from earlier phases of the
project.

These files may refer to the product as `Kingston Care Connect`, use older
domains, or describe superseded deployment paths. Those references are kept for
historical accuracy.

For current naming and operational guidance:

- Use `HelpBridge` as the product name.
- Use `https://helpbridge.ca` as the canonical public domain.
- Prefer the active documentation outside this archive.
